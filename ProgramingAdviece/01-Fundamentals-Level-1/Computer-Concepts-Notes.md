## 4Ô∏è‚É£ How a Computer Executes a Program

1. A programmer writes source code using a high-level language.
2. The compiler translates it into machine code.
3. The executable file (.exe) is stored on secondary storage (Hard Disk).
4. When the program runs:
   - The operating system loads it into RAM.
   - The CPU fetches instructions from memory.
   - The Control Unit decodes instructions.
   - The ALU performs calculations.
   - Results are stored back in memory.
   - Output is displayed to the user.

This process is called the Fetch‚ÄìDecode‚ÄìExecute cycle.

## 6Ô∏è‚É£ Why Computers Use Binary

Computers use binary because transistors inside hardware have only two states:

- ON (1)
- OFF (0)

Electrical voltage levels represent these states.

This makes binary the most reliable system for electronic circuits.
